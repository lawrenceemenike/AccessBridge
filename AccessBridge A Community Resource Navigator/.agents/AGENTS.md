# Global Engineering Standards

- **Zero-Trust Security**: All inputs must be validated and sanitized. Do not trust any external or internal entity by default.
- **Spec-Driven Development (SDD)**: All features must be defined in Markdown with nested YAML schemas before implementation.
- **MCP Tool Execution**: All interactions with external systems must occur through defined Model Context Protocol (MCP) servers.
- **OpenTelemetry Observability**: All components must be instrumented with OpenTelemetry for tracing, metrics, and logging.
